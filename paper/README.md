# Paper

`main.tex` — ACM `sigconf` (`acmart`). `references.bib` — bibliography.
`figures/` — vector PDFs (with PNG previews) generated by
`src/figures.py` and `src/figures_betting.py`.

## Build

```bash
latexmk -pdf main.tex        # or: pdflatex; bibtex; pdflatex; pdflatex
```

Needs a TeX distribution with the `acmart` class (TeX Live / MiKTeX).

## Before submitting

- Fill in the **author affiliation** (line ~19) and, for a real submission,
  remove `nonacm` and add the venue's rights/DOI block.
- Confirm the **ChatGPT minor version** (footnote in §3) — recorded as
  GPT-5.5 (Thinking, high), the ChatGPT high-reasoning model at the 11 Jun 2026
  kickoff, held constant.
- Regenerate figures if the data changes: `python src/run_all.py`.
